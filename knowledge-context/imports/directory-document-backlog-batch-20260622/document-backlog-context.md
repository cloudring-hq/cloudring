# Directory Document Backlog Context

This run adds structural coverage for documents whose raw text was not safely extracted in the document batch. It records container/page/slide/sheet/media/macro/embedded-object structure only, without copying raw document text, source paths, file names, internal member names, metadata values, domains, provider names, people, brand names, or secrets.

## Agent Use

- Use `document-backlog-cases.jsonl` for structural evidence from large, failed, encrypted, and legacy documents.
- Use `document-backlog-signals.jsonl` for stable-id traceability and controlled structure buckets only.
- Treat structural probe failures as retained backlog, not as dropped experience.
- Route macro, embedded-object, encrypted, and sensitive signals to specialized safe handling.
