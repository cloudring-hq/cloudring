# Conformance Profiles

Эта папка хранит readiness profiles для законченных продуктовых этапов
CloudRING. Каждый профиль отвечает на вопрос: какие доказательства нужны, чтобы
AI-агент, человек и будущая команда могли честно сказать, что stage готов в
заявленном scope.
